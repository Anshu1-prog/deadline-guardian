# 🚀 Deadline Guardian – AI Productivity Companion

Deadline Guardian is an AI-powered productivity companion built with the Gemini API that helps users transform goals into structured, actionable execution roadmaps. Instead of relying on passive reminders, it actively assists users in planning, prioritizing, and completing tasks before deadlines.

---

## 📖 Problem Statement

**The Last-Minute Life Saver**

Students, professionals, and entrepreneurs often miss deadlines because existing productivity tools only provide reminders rather than helping users take meaningful action. Deadline Guardian addresses this challenge by acting as an AI planning assistant that guides users toward completing their goals.

---

## 💡 Solution Overview

Deadline Guardian uses Google's Gemini API to analyze a user's objective and generate a personalized execution roadmap. The application considers deadlines, workload tolerance, and experience level to create structured plans that are practical and easy to follow.

---

## ✨ Key Features

- 🤖 AI-powered roadmap generation using Gemini API
- 📅 Deadline-aware task planning
- 🎯 Intelligent task prioritization
- 📚 Skill-level based planning
- ⚡ Workload-aware scheduling
- 🔄 Roadmap rescheduling
- 📊 Interactive roadmap visualization
- 🎨 Modern responsive interface

---

## ☁️ Google Technologies Used

- **Gemini API (Google AI Studio)** – AI reasoning and roadmap generation
- **Google Cloud Run** – Application deployment
- **Google Cloud Build** – Automated build and deployment
- **Google AI Studio** – Prompt development and testing

The project architecture also includes integration points for Google Calendar and Gmail services. Due to the one-week hackathon timeline, complete OAuth authentication could not be finalized before submission, although the integration logic is already incorporated into the codebase.
